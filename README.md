# Togeeat backend API

## Description
Backend API of Togeeat web app

## Installation
- Dependencies:
```bash
$ yarn install
```
- Fill environment fields in '.env' file

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
